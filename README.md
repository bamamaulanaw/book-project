# Book Project

This project is a web-based digital library application that allows users to search for books using the Google Books API.

## Description

Users can view book details such as title, author, publication year, rating, and description through an interactive interface.

## Technologies 

- HTML5
- CSS3
- JavaScript
- Google Books API
- Font Awesome
- Google Fonts

## Features

- Search books by keyword
- Display book results in a responsive grid
- View detailed book information in a modal popup
- Loading indicator during data fetch
- Error handling for empty or failed searches

## Installation

1. Clone this repository:
    ```bash
    git clone https://github.com/bamamaulanaw/book-project.git
    ```
2. Open `index.html` in your browser.

## Usage

Enter a keyword in the search bar to find books. Click on a book card to view detailed information.
